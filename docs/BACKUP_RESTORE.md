# 💾 النسخ الاحتياطي والاستعادة — Bunun Mazen Pharmacy

## النسخ الاحتياطي التلقائي

### التكرار
- كل 24 ساعة تلقائياً
- يتحقق عند كل إقلاع (إذا مرّت 24 ساعة+ منذ آخر نسخة)

### الموقع
- `Desktop/PharmacyBackups/auto_backup_<timestamp>.enc`
- يحتفظ بآخر 7 نسخ (يحذف الأقدم)

### التشفير
- **الخوارزمية:** AES-256-GCM
- **مفتاح الاشتقاق:** PBKDF2-HMAC-SHA256 (100,000 تكرار)
- **كلمة المرور:** تُولَّد عشوائياً (32 بايت) وتُخزَّن في:
  - Linux: `~/.local/share/BununMazenPharmacy/.backup_key`
  - Windows: `%LOCALAPPDATA%\BununMazenPharmacy\.backup_key`
  - macOS: `~/Library/Application Support/BununMazenPharmacy/.backup_key`
- **صلاحيات:** 0600 على Unix (للمالك فقط)

### البيانات المُنسوخة
- medicines, medicine_batches, invoices, invoice_items
- expenses, customer_debts, suppliers, patients
- settings, audit_logs

## النسخ الاحتياطي اليدوي

1. اذهب إلى **الإعدادات > النسخ الاحتياطي**
2. أدخل كلمة مرور قوية (تُستخدم للتشفير)
3. اضغط **إنشاء نسخة احتياطية**
4. الملف يُحفظ على سطح المكتب

## الاستعادة

### من واجهة التطبيق
1. اذهب إلى **الإعدادات > النسخ الاحتياطي**
2. اختر ملف النسخة (.enc)
3. أدخل كلمة المرور
4. اضغط **استعادة**
5. **أعد تشغيل التطبيق** لرؤية البيانات المستعادة

### من سطر الأوامر
```bash
# فك تشفير النسخة (للمعاينة)
cargo run --bin restore_backup -- <file_path> <password>

# استعادة كاملة لقاعدة البيانات
# (يتم من واجهة التطبيق فقط — تتطلب جلسة مدير)
```

## أفضل الممارسات

1. **انقل النسخ الاحتياطية** إلى تخزين خارجي (USB، سحابة) أسبوعياً
2. **اختبر الاستعادة** شهرياً على جهاز اختبار
3. **لا تحذف** ملف `.backup_key` — بدونها لا يمكن فك تشفير النسخ التلقائية
4. **غيّر كلمة مرور النسخ اليدوية** بانتظام
5. **تحقق من حجم** النسخ — إذا كان صغيراً جداً قد يكون تالفاً
